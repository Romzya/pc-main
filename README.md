# PC Builder

Веб-приложение для сборки компьютеров с проверкой совместимости и админ-панелью.

## Функции
- Регистрация и авторизация пользователей
- Создание и управление сборками ПК
- Проверка совместимости комплектующих
- Чат с администраторами
- Админ-панель с полным управлением
- Каталог комплектующих
- Автоматический расчет стоимости

## Технологии
- Python 3.8+
- Flask
- SQLite
- HTML/CSS/JavaScript

## Установка

1. Клонируйте репозиторий:
```bash
git clone [ссылка]
cd pc-builder
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте окружение:
- Windows: `venv\Scripts\activate`
- Linux/Mac: `source venv/bin/activate`

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Запустите приложение:
```bash
python app.py
```

Приложение будет доступно по адресу: `http://localhost:5125`

## Тестовые учетные записи

**Администратор:**
- Логин: `admin`
- Пароль: `admin123`

**Обычный пользователь:** Создается при регистрации

## Структура проекта
- `app.py` - основное приложение
- `db.db` - база данных SQLite
- `templates/` - HTML шаблоны
- `static/` - статические файлы

## База данных
- users - пользователи
- user_gadjets - сборки
- components - комплектующие
- build_chat - сообщения чата
- build_status - статусы сборок

## API Endpoints
- `GET /api/components` - все комплектующие
- `GET /api/components/<category>` - комплектующие по категории
- `GET /api/check_compatibility` - проверка совместимости
